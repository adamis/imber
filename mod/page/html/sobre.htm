Projeto Imber

Este projeto surgiu através de um projeto de iniciação cientÍfica da Universidade de Uberaba (UNIUBE), pelos alunos Adamis Starling da Rocha e Ricael Spirandelli Rocha, onde foi proposto criar um sistema de aquisição de dados e uma pequena estação meteriológica, de forma a ser o mais acessível financeiramente e o mais fácil de ser montar possível.
Foi construído com o intuito de receber todos os dados da estação em tempo real e armazenar e organizar de forma a posteriomente mostrar em gráficos.
Além de gráficos, o Imber disponibiliza um WebService para o consumo destas informações colhidas em forma de JSON (JavaScript Object Notation), assim, facilitando a extração de dados na sua base.
